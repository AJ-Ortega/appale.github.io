Ejercicio 1

  1. ¿Cuál es la función de la URL y URI?
    R//El URI identifica un recurso y lo diferencia de otros mediante un nombre, una ubicación o ambos. La URL identifica la dirección web o la ubicación de un recurso único.


  2. ¿Qué es la Web?
    R//Es un sistema interconectado de páginas web públicas accesibles a través de Internet. La Web no es lo mismo que el Internet: la Web es una de las muchas aplicaciones construidas sobre Internet.


  3. ¿Quién inventó la World Wide Web?
    R//La Web se desarrolló entre mayo de 1989 y diciembre de 1990​​ por el inglés Tim Berners-Lee y con la ayuda del belga Robert Cailliau, mientras ambos trabajaban en el CERN en Ginebra, Suiza, y publicado en 1991.


  4. Por medio de un diagrama indique cómo funciona la web
    R//Un diagrama de flujo es un diagrama que describe un proceso, sistema o algoritmo informático. Se usan ampliamente en numerosos campos para documentar, estudiar, planificar, mejorar y comunicar procesos que suelen ser complejos en diagramas claros y fáciles de comprender. Los diagramas de flujo emplean rectángulos, óvalos, diamantes y otras numerosas figuras para definir el tipo de paso, junto con flechas conectoras que establecen el flujo y la secuencia.


  5. Qué características tiene un Cliente Web y un Servidor Web
    1.Necesitan un soporte físico: Los servidores web deben tener un hardware, que es el equipo que resguarda la información que será transmitida al ser solicitada por los usuarios.

    2.Requieren un sistema operativo: Para que la información del hardware pueda ser extraída y direccionada a su destino final, es esencial tener un software diseñado para llevar a cabo estas tareas.

    3.Cuentan con un Protocolo de Transferencia de Hipertexto: También conocido como HTTP. Son instrucciones que permiten la transmisión de información. Los HTTP establecen la comunicación entre navegadores o plataformas de búsqueda con las bases de datos de los servidores web.
