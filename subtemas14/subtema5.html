World Wide Web ¿Cómo se inició?

  1. Tim Barnes Lee (CERN, 1989).

  2. Repositorio estático de documentos:

    -Una forma cómoda de acceder a documentos estáticos interrelacionados almacenadosen servidores distribuidos alrededor del mundo.

  3. La especificación inicial incluía:

    -Un lenguaje para dar formato a los documentos (HTML).

    -Un protocolo sencillo para comunicar navegadores (clientes) y servidores (HTTP).
